About This Project

This Angular application was created for Lab 4 and demonstrates standalone components, routing, signal-based inputs and outputs, and modern control flow syntax.

  • Data model: strongly typed Product interface
  • Reusable UI: separate product list and product card components
  • Dynamic rendering: @for and @if
  • Sharing: WhatsApp and Telegram links generated with encodeURIComponent()
